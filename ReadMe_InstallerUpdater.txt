NOTE: You will need Wine or Proton/Steam to play the game on Linux. This script only downloads/updates the game.

--- INSTRUCTIONS ---
1. Give INSTALL_OR_UPDATE.sh executable permissions by Right-Click -> Properties -> Permissions
2. Open INSTALL_OR_UPDATE.sh
-OR-
1. Open a terminal where INSTALL_OR_UPDATE.sh is located and enter these commands:
chmod +x ./INSTALL_OR_UPDATE.sh
./INSTALL_OR_UPDATE.sh

--- STEAM DECK INSTRUCTIONS ---
1. Right-click INSTALL_OR_UPDATE.sh and select "Properties"
2. Select the "Permissions" tab
3. Check the "Is Executable" checkbox and hit OK
4. Right-click INSTALL_OR_UPDATE.sh and select "Run In Konsole"
5. In Steam, select "Add a Game" -> "Add a Non-Steam Game"
6. Add the "Game.exe" from your InfiniteFusion folder
