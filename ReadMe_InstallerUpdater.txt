--- INSTRUCTIONS ---
1. Run "INSTALL_OR_UPDATE.bat" to download the game. 
2. When a new version of the game releases, run "INSTALL_OR_UPDATE.bat" again.

--- TO UPDATE AN OLD VERSION OF THE GAME THAT WAS DOWNLOADED WITHOUT "INSTALL_OR_UPDATE.bat" ---
1. Drag "INSTALL_OR_UPDATE.bat" and "REQUIRED_BY_INSTALLER_UPDATER" into your game's root directory (Where Game.exe is located.)
2. Run "INSTALL_OR_UPDATE.bat".
